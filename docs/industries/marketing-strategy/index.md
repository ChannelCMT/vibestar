# 营销策略的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 06</p>
  <h2>从脑暴方案，到可测试的营销增长实验。</h2>
  <span>活动方案 / 用户洞察 / 广告文案 / 落地页 / 投放复盘</span>
</section>

营销策略不再只是写一个漂亮方案，而是要快速找到用户洞察、生成创意、搭建落地页、测试转化并复盘结果。Vibe Coding 让策略从“提案型工作”升级为“实验型工作”。

## 典型任务

<section class="industry-tasks">
  <article><h3>用户洞察</h3><p>分析目标人群、使用场景、购买动机和阻碍因素。</p></article>
  <article><h3>创意方案</h3><p>生成活动主题、传播话术、广告文案和渠道策略。</p></article>
  <article><h3>增长实验</h3><p>制作落地页、设计 A/B 测试、整理投放结果和优化建议。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>靠经验脑暴、手动写方案、做 PPT，再等待设计和开发资源。</p><ul><li>创意验证慢</li><li>用户洞察不够系统</li><li>落地页制作周期长</li><li>复盘容易停留在表面数据</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 生成用户画像、活动方案、文案变体和落地页结构。</p><ul><li>快速拆解目标用户</li><li>生成多套活动主题</li><li>批量测试广告文案</li><li>自动整理投放复盘</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用洞察、创意、页面、投放和复盘 Agent 形成增长实验链路。</p><ul><li>洞察 Agent 分析人群</li><li>创意 Agent 生成方案</li><li>落地页 Agent 生成页面</li><li>复盘 Agent 给出优化动作</li></ul></article>
</section>

## 一个策略人的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>定义目标</h3><p>明确活动目的、目标人群、渠道和转化指标。</p></div>
  <div><span>Step 2</span><h3>生成方案</h3><p>让 AI 输出洞察、核心主张、创意方向和文案。</p></div>
  <div><span>Step 3</span><h3>搭建落地页</h3><p>用 Vibe Coding 生成页面结构和可预览 Demo。</p></div>
  <div><span>Step 4</span><h3>复盘优化</h3><p>将投放数据交给 Agent，总结问题并提出下一轮实验。</p></div>
</section>

## 课堂练习

你将选择一个产品或服务，完成：

- 一份营销策略 brief
- 三套广告文案
- 一个活动落地页
- 一个 A/B 测试方案
- 一份投放复盘框架

## 学完后你能做到

<section class="industry-outcomes">
