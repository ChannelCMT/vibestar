# 任何人的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 14</p>
  <h2>把生活和工作中的具体问题，拆成 AI 可以执行的步骤。</h2>
  <span>个人效率 / 家庭事务 / 学习计划 / 小工具</span>
</section>

Vibe Coding 不只属于程序员，也不只属于职场专家。任何人只要有一个具体问题，都可以学习如何把目标说清楚、把任务拆开、让 AI 生成方案，并一步步做出可运行的小工具。

## 典型任务

<section class="industry-tasks">
  <article><h3>个人效率</h3><p>整理计划、待办、资料、笔记和复盘流程。</p></article>
  <article><h3>生活工具</h3><p>制作预算表、旅行计划、家庭清单、学习打卡和提醒工具。</p></article>
  <article><h3>问题解决</h3><p>把复杂问题拆成步骤，生成执行清单和检查标准。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>遇到问题靠搜索、复制模板、手动整理和反复试错。</p><ul><li>信息分散</li><li>执行靠记忆</li><li>模板不贴合自己</li><li>很多想法停在脑子里</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 帮你规划、整理、生成和检查结果。</p><ul><li>生成个性化计划</li><li>整理资料和清单</li><li>制作简单网页或脚本</li><li>解释不懂的概念</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>把长期任务拆给不同 Agent，形成个人工作系统。</p><ul><li>计划 Agent 拆任务</li><li>执行 Agent 生成内容</li><li>提醒 Agent 跟踪进度</li><li>复盘 Agent 优化习惯</li></ul></article>
</section>

## 一个普通人的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>提出问题</h3><p>把生活或工作中的烦恼写成清晰目标。</p></div>
  <div><span>Step 2</span><h3>拆解任务</h3><p>让 AI 把目标拆成步骤、材料、时间和检查点。</p></div>
  <div><span>Step 3</span><h3>生成工具</h3><p>制作一个表格、页面、脚本或自动化清单。</p></div>
  <div><span>Step 4</span><h3>持续使用</h3><p>根据实际效果让 AI 调整流程，让工具越来越贴合自己。</p></div>
</section>

## 课堂练习

- 一个个人效率工具
- 一份自动化清单
- 一个资料整理流程
- 一份长期计划模板
- 一份复盘和改进记录
