# 企业咨询的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 10</p>
  <h2>从资料堆叠，到快速形成诊断和方案。</h2>
  <span>行业研究 / 访谈纪要 / 诊断报告 / 方案框架</span>
</section>

咨询工作的难点在于信息量大、结构化要求高、交付节奏快。Vibe Coding 能帮助咨询顾问把资料搜集、访谈整理、问题诊断、方案生成和报告制作连接成一条可复用的交付流水线。

## 典型任务

<section class="industry-tasks">
  <article><h3>行业研究</h3><p>整理行业趋势、竞品案例、市场格局和关键数据。</p></article>
  <article><h3>问题诊断</h3><p>从访谈、流程和数据中提炼症结、假设和改进方向。</p></article>
  <article><h3>方案交付</h3><p>生成报告框架、行动计划、路线图和客户汇报材料。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>人工查资料、做访谈、整理纪要、搭框架和写报告。</p><ul><li>资料处理量大</li><li>访谈信息难结构化</li><li>报告初稿耗时</li><li>方案推演依赖个人经验</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 总结访谈、提炼问题、生成框架和改写报告。</p><ul><li>快速归纳材料</li><li>生成诊断假设</li><li>输出报告结构</li><li>模拟客户质询</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用研究、访谈、诊断、方案和报告 Agent 协作交付。</p><ul><li>研究 Agent 做行业扫描</li><li>访谈 Agent 整理纪要</li><li>诊断 Agent 提炼问题</li><li>报告 Agent 生成材料</li></ul></article>
</section>

## 一个咨询顾问的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>定义课题</h3><p>明确客户背景、咨询目标、交付范围和关键问题。</p></div>
  <div><span>Step 2</span><h3>整理资料</h3><p>让 AI 归纳行业材料、访谈纪要和数据线索。</p></div>
  <div><span>Step 3</span><h3>形成诊断</h3><p>生成问题树、假设清单和优先级判断。</p></div>
  <div><span>Step 4</span><h3>输出方案</h3><p>生成报告框架、行动路线图和客户汇报稿。</p></div>
</section>
