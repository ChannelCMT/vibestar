# 企业客服的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 07</p>
  <h2>从人工回复，到搭建知识库和智能分流系统。</h2>
  <span>知识库问答 / 客户分流 / 话术生成 / 问题归因</span>
</section>

客服工作中大量问题是重复的，但每个客户的语气、背景和紧急程度又不同。Vibe Coding 能帮助客服团队把经验沉淀成知识库，把重复回复变成自动化流程，把问题归因变成可改进的数据。

## 典型任务

<section class="industry-tasks">
  <article><h3>知识沉淀</h3><p>把产品文档、历史工单和常见问题整理成可检索知识库。</p></article>
  <article><h3>智能回复</h3><p>生成不同语气、场景和风险等级的回复话术。</p></article>
  <article><h3>问题归因</h3><p>分析工单类型、客户痛点和流程缺陷，推动产品和运营改进。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>人工查资料、复制话术、手动分类问题和统计表格。</p><ul><li>重复问题占用大量时间</li><li>回复质量依赖个人经验</li><li>知识库更新滞后</li><li>问题归因不够系统</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 总结工单、生成回复、改写语气和整理 FAQ。</p><ul><li>快速生成客户回复</li><li>自动归纳高频问题</li><li>根据客户情绪调整表达</li><li>输出客服日报和周报</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用分流、问答、质检和归因 Agent 构建客服系统。</p><ul><li>分流 Agent 判断问题类型</li><li>问答 Agent 生成回复草稿</li><li>质检 Agent 检查合规风险</li><li>归因 Agent 输出改进建议</li></ul></article>
</section>

## 一个客服团队的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>整理知识</h3><p>导入常见问题、产品说明和历史工单。</p></div>
  <div><span>Step 2</span><h3>生成话术</h3><p>让 AI 生成标准回复、安抚话术和升级规则。</p></div>
  <div><span>Step 3</span><h3>搭建分流</h3><p>根据问题类型、紧急程度和客户状态设计分流流程。</p></div>
  <div><span>Step 4</span><h3>复盘归因</h3><p>让 Agent 分析工单数据，发现产品和流程问题。</p></div>
</section>

## 课堂练习

你将选择一个产品客服场景，完成：

- 一个 FAQ 知识库
- 一套标准回复话术
- 一个问题分流流程
- 一份客服质检规则
- 一份工单归因报告

## 学完后你能做到

<section class="industry-outcomes">
  <div><h3>减少重复回复</h3><p>把高频问题沉淀成知识库和自动化话术。</p></div>
  <div><h3>提升服务稳定性</h3><p>用质检 Agent 检查语气、事实和合规风险。</p></div>
  <div><h3>推动业务改进</h3><p>把客服问题转化成产品、运营和流程优化线索。</p></div>
</section>
