# 投资理财的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 12</p>
  <h2>从资料阅读，到建立可追踪的研究流程。</h2>
  <span>资料整理 / 研报摘要 / 标的跟踪 / 风险提示 / 投资逻辑表达</span>
</section>

投资理财工作面对的是大量信息、快速变化和持续跟踪。Vibe Coding 不是替你做投资决策，而是帮助你更高效地整理资料、形成研究框架、追踪变化和表达投资逻辑。

## 典型任务

<section class="industry-tasks">
  <article><h3>资料整理</h3><p>汇总研报、公告、新闻、财报和行业资料，提炼关键信息。</p></article>
  <article><h3>逻辑分析</h3><p>梳理商业模式、增长驱动、风险因素和关键假设。</p></article>
  <article><h3>持续跟踪</h3><p>建立标的跟踪表、事件提醒、风险提示和复盘记录。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>人工读研报、摘重点、整理表格、写观点和跟踪变化。</p><ul><li>信息量大，阅读压力高</li><li>观点散落在不同文档</li><li>风险提示容易遗漏</li><li>跟踪过程不连续</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 摘要资料、提炼风险、生成跟踪清单和表达框架。</p><ul><li>快速摘要研报和公告</li><li>梳理投资逻辑和反方观点</li><li>生成风险提示清单</li><li>辅助写研究笔记</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用资料、跟踪、风险和报告 Agent 建立研究管线。</p><ul><li>资料 Agent 摘要信息</li><li>逻辑 Agent 梳理假设</li><li>风险 Agent 提醒异常</li><li>报告 Agent 生成研究记录</li></ul></article>
</section>

## 一个投资研究者的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>导入资料</h3><p>整理研报、公告、新闻或财务数据。</p></div>
  <div><span>Step 2</span><h3>提炼逻辑</h3><p>让 AI 输出核心观点、关键假设和风险因素。</p></div>
  <div><span>Step 3</span><h3>建立跟踪</h3><p>生成标的跟踪表、事件清单和观察指标。</p></div>
  <div><span>Step 4</span><h3>复盘更新</h3><p>根据新信息让 Agent 更新判断依据和风险提示。</p></div>
</section>

<section class="industry-outcomes">
