# 销售顾问的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 08</p>
  <h2>从跟进客户，到系统化提升转化效率。</h2>
  <span>客户画像 / 跟进话术 / 方案提案 / 成交复盘</span>
</section>

销售顾问的核心不是“多发几条消息”，而是理解客户处境、抓住决策动机、设计沟通节奏，并把每次跟进沉淀成可复用经验。Vibe Coding 能帮助销售把客户研究、话术生成、方案制作和成交复盘串成一个系统。

## 典型任务

<section class="industry-tasks">
  <article><h3>客户洞察</h3><p>整理客户背景、行业痛点、决策链路和潜在阻碍。</p></article>
  <article><h3>沟通推进</h3><p>生成初次触达、需求确认、异议处理和二次跟进话术。</p></article>
  <article><h3>方案成交</h3><p>输出定制化提案、报价说明、会议纪要和成交复盘。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>手动查客户资料，凭经验写话术，靠记忆管理跟进节奏。</p><ul><li>客户画像不完整</li><li>跟进内容容易同质化</li><li>提案制作耗时</li><li>成交复盘难沉淀</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 生成客户画像、沟通脚本、提案框架和复盘总结。</p><ul><li>快速整理客户背景</li><li>生成不同阶段话术</li><li>根据异议生成回应策略</li><li>自动整理会议纪要和下一步</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用研究、话术、提案和复盘 Agent 协作推动销售流程。</p><ul><li>研究 Agent 建客户画像</li><li>话术 Agent 生成沟通脚本</li><li>提案 Agent 输出方案材料</li><li>复盘 Agent 总结成交规律</li></ul></article>
</section>

## 一个销售顾问的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>建立画像</h3><p>输入客户行业、规模、角色、需求和历史沟通记录。</p></div>
  <div><span>Step 2</span><h3>设计话术</h3><p>让 AI 输出初访、跟进、异议处理和成交推进话术。</p></div>
  <div><span>Step 3</span><h3>生成提案</h3><p>根据客户痛点生成方案结构、价值点和演示材料。</p></div>
  <div><span>Step 4</span><h3>复盘迭代</h3><p>把成交或流失过程交给 Agent 分析，形成下一轮策略。</p></div>
</section>

## 课堂练习

- 一份个性化销售提案
- 一份会议纪要模板
- 一份成交复盘表
