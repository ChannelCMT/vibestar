# 教育培训的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 11</p>
  <h2>从备课到智能学习工具，提升教学效率。</h2>
  <span>课程脚本 / 课件 / 练习题 / 自动问答 / 个性化学习</span>
</section>

教育培训的核心不是把知识讲完，而是让不同基础的学习者真正理解、练习、反馈和进步。Vibe Coding 能帮助教师和培训者快速生成教学内容、练习系统、答疑工具和个性化学习反馈。

## 典型任务

<section class="industry-tasks">
  <article><h3>课程设计</h3><p>生成课程大纲、教学目标、案例脚本和课堂活动。</p></article>
  <article><h3>练习评测</h3><p>生成题目、答案解析、评分标准和分层练习。</p></article>
  <article><h3>学习支持</h3><p>搭建自动问答、学习计划、错题整理和个性化反馈工具。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>手动备课、做课件、出题、批改和答疑，所有环节都依赖人工。</p><ul><li>备课材料制作耗时</li><li>题目难以快速分层</li><li>个性化反馈不足</li><li>学生问题难沉淀</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 生成课程脚本、案例、练习题和答疑材料。</p><ul><li>快速生成课程结构</li><li>按难度生成练习</li><li>把复杂概念换成通俗解释</li><li>生成课堂互动问题</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>用教案、题库、答疑、评测和反馈 Agent 构建学习系统。</p><ul><li>教案 Agent 设计课程</li><li>题库 Agent 生成练习</li><li>答疑 Agent 回答常见问题</li><li>评测 Agent 输出学习建议</li></ul></article>
</section>

## 一个教育培训者的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>定义学习目标</h3><p>明确学员画像、课程目标、知识点和学习成果。</p></div>
  <div><span>Step 2</span><h3>生成课程内容</h3><p>让 AI 输出课程大纲、案例、讲稿和课堂练习。</p></div>
  <div><span>Step 3</span><h3>搭建学习工具</h3><p>生成答疑页面、练习题库或学习计划工具。</p></div>
  <div><span>Step 4</span><h3>反馈迭代</h3><p>根据作业和测评结果，让 Agent 给出个性化学习建议。</p></div>
</section>

- 一套分层练习题
