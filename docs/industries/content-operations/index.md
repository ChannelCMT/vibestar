# 新媒体运营的 Vibe Coding 工作流

<section class="industry-hero">
  <p>Industry Scenario 05</p>
  <h2>从追热点，到搭建持续产出的内容系统。</h2>
  <span>选题 / 脚本 / 素材 / 发布 / 复盘</span>
</section>

新媒体运营的压力不只是“写一篇内容”，而是持续找到选题、适配平台、生成素材、发布测试并复盘数据。Vibe Coding 能把内容生产从临时灵感，升级成可复用的内容工作流。

## 典型任务

<section class="industry-tasks">
  <article><h3>选题策划</h3><p>根据用户、热点、产品卖点和平台特性生成内容方向。</p></article>
  <article><h3>内容生产</h3><p>生成短视频脚本、图文框架、标题、封面文案和素材提示词。</p></article>
  <article><h3>数据复盘</h3><p>整理阅读、点击、转化、评论和留资数据，形成下一轮优化建议。</p></article>
</section>

## 三种工作流对比

<section class="workflow-grid">
  <article><strong>01</strong><h3>古法工作流程</h3><p>人工刷热点、找案例、写脚本、排期发布，再手动做复盘。</p><ul><li>选题依赖临场感觉</li><li>内容风格不稳定</li><li>素材和文案重复劳动多</li><li>复盘难沉淀成方法</li></ul></article>
  <article><strong>02</strong><h3>对话式 AI 工作流程</h3><p>让 AI 生成选题池、脚本草稿、标题变体和复盘摘要。</p><ul><li>快速扩展选题角度</li><li>批量生成不同平台文案</li><li>根据人设改写语气</li><li>把数据转成复盘结论</li></ul></article>
  <article><strong>03</strong><h3>Agents 工作流</h3><p>让选题、脚本、素材、发布和复盘 Agent 组成内容流水线。</p><ul><li>选题 Agent 监控趋势</li><li>脚本 Agent 生成多版本草稿</li><li>素材 Agent 输出画面建议</li><li>复盘 Agent 提炼增长策略</li></ul></article>
</section>

## 一个运营的实操闭环

<section class="industry-process">
  <div><span>Step 1</span><h3>输入目标</h3><p>明确产品、用户、平台、转化目标和内容约束。</p></div>
  <div><span>Step 2</span><h3>生成选题</h3><p>让 AI 输出选题矩阵、标题方向、内容结构和差异化角度。</p></div>
  <div><span>Step 3</span><h3>生产内容</h3><p>生成脚本、图文、封面文案、素材提示词和发布排期。</p></div>
  <div><span>Step 4</span><h3>复盘迭代</h3><p>把数据交给 Agent 分析，生成下一轮优化建议。</p></div>
</section>

## 课堂练习

你将围绕一个真实账号或产品，完成：

- 一份 7 天选题表
- 三条短视频脚本
- 一组标题和封面文案
- 一个发布排期表
- 一份数据复盘模板

## 学完后你能做到

<section class="industry-outcomes">
  <div><h3>稳定产出内容</h3><p>用 AI 扩展选题和脚本，减少灵感枯竭。</p></div>
  <div><h3>快速适配平台</h3><p>把同一内容改写成适合小红书、视频号、公众号或抖音的表达。</p></div>
  <div><h3>形成增长闭环</h3><p>用复盘 Agent 把数据变成下一轮内容策略。</p></div>
</section>
