# 学习路径

Vibe Star 的目标是帮助学员驾驭 AI，成为能用 AI 重塑工作效能的 Vibe Star。

课程不是单纯介绍工具，而是把学习拆成三段清晰路径：先跑通，再进入工作场景，最后掌握多 Agents 管理与部署服务。

## Level 1: 安装环境，测试跑通

这一阶段解决基础门槛，让每位学员都能稳定使用自己的 AI 工具链。

- 安装和配置必要环境
- 跑通基础模型调用与对话流程
- 建立文件、素材和提示词管理习惯
- 完成第一个可复现的小任务

## Level 2: 线下职业场景化学习

这一阶段围绕不同职业角色展开，把 AI 放进真实工作任务里。

- 运营：活动方案、内容排期、数据复盘
- 产品：需求拆解、竞品分析、PRD 辅助
- 设计：灵感板、视觉方向、交付说明
- 销售：客户画像、话术生成、跟进总结
- 管理：会议纪要、决策辅助、流程优化

## Level 3: 多 Agents 管理与部署服务

这一阶段从个人使用走向系统化能力。

- 拆解复杂任务并分配给不同 Agents
- 管理多 Agents 的输入、输出和协作边界
- 把高频工作流封装为可复用服务
- 学习部署、监控和迭代自己的 AI 工作系统
